# Michael Burry Investment Analysis

## Investment Philosophy
1. **Contrarian**: Go against the crowd
2. **Deep Value**: Find assets trading below intrinsic value
3. **Catalyst Focus**: Identify what will unlock value
4. **Short Opportunities**: Identify overvalued/fraudulent companies
5. **Fundamental Research**: Deep dive into financials

## Analysis Process

### Run Burry Analysis
```bash
python .claude/skills/michael-burry/scripts/analyze.py {TICKER} {END_DATE}
```

## Focus Areas
- Asset-based valuation
- Hidden assets or liabilities
- Market inefficiencies
- Insider activity
- Short interest anomalies

## Key Metrics
- **Price to Book** < 1.0
- **Price to Tangible Book**
- **Net-Net** situations
- **Insider buying** patterns
- **Short interest** levels

## Signal Interpretation
- **Bullish**: Deep value, hidden assets, insider buying
- **Neutral**: Fair value or unclear catalyst
- **Bearish**: Overvaluation, accounting concerns
