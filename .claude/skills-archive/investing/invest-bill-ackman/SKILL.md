# Bill Ackman Investment Analysis

## Investment Philosophy
1. **Activist Investing**: Engage to unlock value
2. **Concentrated Positions**: 8-12 positions max
3. **Simple, Predictable Businesses**: High certainty
4. **Strong Management**: Or ability to influence change
5. **Asymmetric Payoffs**: Limited downside, large upside

## Analysis Process

### Run Ackman Analysis
```bash
python .claude/skills/bill-ackman/scripts/analyze.py {TICKER} {END_DATE}
```

## Focus Areas
- Undervalued assets or operations
- Poor capital allocation (fixable)
- Corporate governance issues
- Hidden value to unlock
- Management change potential

## Key Criteria
- **Predictable Cash Flows**: Subscription/recurring
- **Strong Brands**: Pricing power
- **Fixable Problems**: Clear path to value
- **Margin of Safety**: Significant discount

## Signal Interpretation
- **Bullish**: Hidden value, fixable issues, quality business
- **Neutral**: Some value but unclear catalyst
- **Bearish**: No clear path to value creation
